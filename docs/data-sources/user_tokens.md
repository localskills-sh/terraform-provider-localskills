# localskills_user_tokens (Data Source)

Lists all API tokens belonging to the authenticated user. Token secret values are not included in the response; only metadata such as name, creation date, and last-used timestamp is returned.

Use this data source to audit existing tokens, check expiration dates, or verify token inventory before creating new ones.

## Example Usage

```terraform
data "localskills_user_tokens" "all" {}

output "token_names" {
  value = [for t in data.localskills_user_tokens.all.tokens : t.name]
}
```
