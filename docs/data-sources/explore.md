# localskills_explore (Data Source)

Browses publicly available skills on localskills.sh with search, tag filtering, type filtering, and sorting options. Sort by newest, recently updated, or most downloads.

Use this data source to discover community skills, find popular rules, or search for skills by keyword. Results include author information and download counts.

## Example Usage

```terraform
# Browse the most downloaded public skills
data "localskills_explore" "popular" {
  sort = "downloads"
  type = "rule"
}

# Search for skills by keyword
data "localskills_explore" "search" {
  query = "terraform"
  sort  = "newest"
}

output "popular_skills" {
  value = [for s in data.localskills_explore.popular.skills : s.name]
}
```
