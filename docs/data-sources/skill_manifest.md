# localskills_skill_manifest (Data Source)

Retrieves the package manifest for a package-format skill (ZIP archives with multiple files). Returns the manifest name, description, version, and list of files in the package.

Use this data source to inspect the contents of a multi-file skill package or verify its structure before deployment.

## Example Usage

```terraform
data "localskills_skill_manifest" "package" {
  skill_id = localskills_skill.eslint_rules.id
}

output "manifest_files" {
  value = data.localskills_skill_manifest.package.files
}
```
