# localskills_skill_analytics (Data Source)

Retrieves download analytics for a skill including total downloads, unique users, and unique IPs.

Use this data source to monitor skill adoption metrics or build dashboards showing skill usage across your organization.

## Example Usage

```terraform
data "localskills_skill_analytics" "eslint" {
  skill_id = localskills_skill.eslint_rules.id
}

output "total_downloads" {
  value = data.localskills_skill_analytics.eslint.total_downloads
}

output "unique_users" {
  value = data.localskills_skill_analytics.eslint.unique_users
}
```
