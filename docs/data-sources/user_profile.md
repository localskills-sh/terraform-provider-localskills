# localskills_user_profile (Data Source)

Retrieves the authenticated user's profile including username, display name, email address, avatar image URL, and bio.

Use this data source to reference the current user's identity in other resources or to display profile information in outputs.

## Example Usage

```terraform
data "localskills_user_profile" "me" {}

output "my_email" {
  value = data.localskills_user_profile.me.email
}

output "my_username" {
  value = data.localskills_user_profile.me.username
}
```
