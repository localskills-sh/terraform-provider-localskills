# localskills_teams (Data Source)

Lists all teams (tenants) accessible to the authenticated user, including the user's role in each team.

Use this data source to enumerate available teams for multi-team configurations or to dynamically reference team IDs based on the authenticated user's memberships.

## Example Usage

```terraform
data "localskills_teams" "all" {}

output "team_names" {
  value = [for t in data.localskills_teams.all.teams : t.name]
}
```
