# localskills_sso_connection (Data Source)

Retrieves the SSO/SAML connection configuration for a team, including the IdP and SP metadata, enabled status, default role, and email domain restrictions.

Use this data source to inspect SSO settings, retrieve Service Provider URLs for IdP configuration, or verify that SSO is correctly enabled for a team.

## Example Usage

```terraform
data "localskills_sso_connection" "current" {
  tenant_id = localskills_team.engineering.id
}

output "sso_enabled" {
  value = data.localskills_sso_connection.current.enabled
}

output "sp_acs_url" {
  value = data.localskills_sso_connection.current.sp_acs_url
}
```
