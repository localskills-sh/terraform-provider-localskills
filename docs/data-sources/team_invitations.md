# localskills_team_invitations (Data Source)

Lists all invitations for a team, including pending, accepted, and expired invitations. Each invitation includes the email, role, and expiration details.

Use this data source to audit outstanding invitations, verify invitation status, or build reports on team onboarding progress.

## Example Usage

```terraform
# List pending invitations for a team
data "localskills_team_invitations" "pending" {
  tenant_id = localskills_team.engineering.id
}

output "pending_emails" {
  value = [for inv in data.localskills_team_invitations.pending.invitations : inv.email]
}
```
