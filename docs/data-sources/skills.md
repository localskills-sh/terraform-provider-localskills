# localskills_skills (Data Source)

Lists skills with optional filters for tenant, visibility, type, search query, and tags. Returns a list of skills matching the specified criteria.

Use this data source to discover skills across teams, audit skill inventories, or find skills by tag or type. All filter parameters are optional; omitting all filters returns all accessible skills.

## Example Usage

```terraform
data "localskills_skills" "public_rules" {
  tenant_id  = localskills_team.engineering.id
  visibility = "public"
  type       = "rule"
}
```
