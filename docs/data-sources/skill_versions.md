# localskills_skill_versions (Data Source)

Lists all published versions of a skill, including version numbers, semantic versions, content hashes, and creation timestamps.

Use this data source to audit version history, find specific releases, or determine the latest version of a skill. Each version entry includes the commit message and file count.

## Example Usage

```terraform
data "localskills_skill_versions" "history" {
  skill_id = localskills_skill.eslint_rules.id
}
```
